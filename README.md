# Skin Tracker

A mobile-first skincare routine tracker with daily guide, photo progress, before/after compare, weekly log and progress stats.

## Deploy (same as your other apps)

1. Push this folder to a new GitHub repo
2. Go to Vercel → New Project → Import that repo
3. Framework: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`
6. Deploy

Done. Open the Vercel URL on your phone — photos will work fully.

## Local development

```bash
npm install
npm run dev
```
